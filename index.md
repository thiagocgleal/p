# Política de Privacidade – Beach Play

Última atualização: 20 de janeiro de 2026

O Beach Play respeita a privacidade dos usuários e está comprometido com a proteção dos dados pessoais.

## Informações coletadas
- Nome, e-mail e foto de perfil (login via Google)
- Localização aproximada
- Dados de uso do aplicativo
- Identificadores de publicidade

## Uso das informações
- Autenticação do usuário
- Exibição de arenas e quadras próximas
- Melhoria da experiência do aplicativo
- Exibição de anúncios

## Compartilhamento
Os dados podem ser compartilhados com:
- Google Authentication
- Google AdMob
- OpenStreetMap / Overpass API

## Segurança
Adotamos medidas técnicas para proteger os dados contra acessos não autorizados.

## Direitos do usuário
O usuário pode solicitar acesso, correção ou exclusão de seus dados a qualquer momento.
